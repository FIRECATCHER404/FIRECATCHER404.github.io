Blockspace

A lightweight 3D sandbox with physics-driven block interactions.

Blockspace is a Java + LWJGL/OpenGL sandbox game focused on building, interactive physics, and just having fun. You can place and erase blocks easily, shoot projectiles at movable blocks, and move through the world in first or third person.

The base game includes world saving/loading with slots, an in-game pause/settings menu, chat support, and a stylized main menu experience. It is designed as a clean, expandable foundation for future features like multiplayer and richer world systems.

  • Option to place blocks in the air or where you're looking
  • Physics updates for dynamic objects and projectiles
  • Save-slot based world persistence
  • First-person and third-person gameplay views
